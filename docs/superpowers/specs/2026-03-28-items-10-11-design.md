# Items 10 & 11: EXEC Spacing Fix and Duplicate Scrollbar Removal

**Date:** 2026-03-28
**Status:** Approved

## Item 10: Remove blank line before [EXEC] commands

### Problem
In `draw_ai_text_with_exec()`, the text segment extracted before `[EXEC]` includes trailing newlines (e.g. `"commentary text:\n"`). When rendered with `DT_WORDBREAK`, the trailing `\n` creates a visible blank line before the purple command. Space after `[/EXEC]` (before next commentary) should be preserved.

### Solution
In `draw_ai_text_with_exec()` at `src/ui/chat_listview.c`, after copying the pre-EXEC segment to `seg`, strip trailing `\n` and `\r` characters before rendering. This collapses the gap between commentary and the command that follows.

The height measurement in `measure_item()` also needs adjustment — it currently measures the full raw text including `[EXEC]`/`[/EXEC]` tags and all newlines. A dedicated measurement function that applies the same newline-stripping logic should be used, or the measurement should call `draw_ai_text_with_exec()` in `DT_CALCRECT` mode.

### Files changed
- `src/ui/chat_listview.c` — trim trailing newlines from pre-EXEC segments in `draw_ai_text_with_exec()`, update height measurement to match

## Item 11: Remove native Windows scrollbar

### Problem
The ChatListView is created with `WS_VSCROLL` which creates a native Windows scrollbar. Although `ShowScrollBar(hwnd, SB_VERT, FALSE)` hides it initially, `SetScrollInfo(lv->hwnd, SB_VERT, &si, TRUE)` in `update_scrollbar()` can re-show it. This causes a white native scrollbar to appear alongside the custom themed scrollbar (csb).

### Solution
1. Remove `WS_VSCROLL` from `CreateWindowExA` style flags at line 384 — prevents native scrollbar from being created
2. Remove the `SetScrollInfo(lv->hwnd, SB_VERT, ...)` call in `update_scrollbar()` — stop updating native scrollbar state
3. Remove the `ShowScrollBar(hwnd, SB_VERT, FALSE)` call in `chat_listview_set_scrollbar()` — no longer needed since native scrollbar isn't created

### Files changed
- `src/ui/chat_listview.c` — remove WS_VSCROLL flag, remove SetScrollInfo call, remove ShowScrollBar call
